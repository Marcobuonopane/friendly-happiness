Installazione e Deploy

Installazione Locale

1. Clone del Repository

git clone https://github.com/your-repo/autentico-v2.2.git cd autentico-v2.2

2. Installazione Dipendenze

npm install # oppure yarn install

3. Avvio Server di Sviluppo

npm run serve # L'app sarà disponibile su http://localhost:3000

Deploy Mobile

1. Build per Produzione

npm run build:mobile

2. Configurazione Capacitor

npx cap init autentico com.marcobuonopane.autentico npx cap add android npx cap add ios

3. Build e Deploy Automatico

# Play Store automatico npm run deploy:playstore # iOS App Store npm run deploy:appstore
